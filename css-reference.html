🎨 CSS Reference

Selectores, layouts, animaciones, variables y todo lo que puedes hacer con CSS.

🎯 Selectores

Los selectores determinan a qué elementos HTML se aplican los estilos.

tipos de selectores
/* ═══ SELECTORES BÁSICOS ═══ */
*               { box-sizing: border-box; }  /* todos los elementos */
p               { color: red; }               /* por etiqueta */
.clase          { color: blue; }              /* por clase */
#id-unico       { color: green; }             /* por ID (único) */
input[type="text"]{ border: 1px solid; }       /* por atributo */

/* ═══ COMBINADORES ═══ */
div p           { }   /* descendiente (cualquier nivel) */
div > p         { }   /* hijo directo */
h2 + p          { }   /* hermano adyacente (inmediato) */
h2 ~ p          { }   /* todos los hermanos siguientes */

/* ═══ AGRUPACIÓN ═══ */
h1, h2, h3      { font-family: sans-serif; }

/* ═══ ESPECIFICIDAD (de mayor a menor) ═══ */
/* inline style > #id > .clase > etiqueta */
#btn.active:hover{ }  /* combinación de selectores */

📦 Box Model

Todo elemento HTML es una caja. El modelo de caja define sus 4 capas.

visualización del box model
margin
border
padding
CONTENT
.caja {
  /* CONTENIDO */
  width: 200px;
  height: 100px;

  /* PADDING — espacio interno */
  padding: 20px;                /* todos lados */
  padding: 10px 20px;           /* vertical horizontal */
  padding: 10px 15px 20px 5px;  /* top right bottom left */

  /* BORDER */
  border: 2px solid #333;
  border-radius: 8px;
  border-top: 3px dashed red;

  /* MARGIN — espacio externo */
  margin: 20px;
  margin: 0 auto;   /* centrar horizontalmente */

  /* box-sizing: cambia cómo se calcula el ancho total */
  box-sizing: border-box;   /* width INCLUYE padding+border */
  box-sizing: content-box;  /* default: width solo del contenido */
}

👁️ Display

La propiedad display controla cómo se comporta un elemento en el flujo del documento.

valores de display
display: block;         /* ocupa todo el ancho (div, p, h1) */
display: inline;        /* solo el ancho del contenido (span, a) */
display: inline-block;  /* inline pero acepta width/height */
display: flex;          /* contenedor flexbox */
display: grid;          /* contenedor grid */
display: none;          /* oculta (no ocupa espacio) */
visibility: hidden;     /* oculta pero SÍ ocupa espacio */
opacity: 0;             /* transparente, ocupa espacio */

.contenedor {
  display: flex;

  /* DIRECCIÓN */
  flex-direction: row;          /* → fila (default) */
  flex-direction: column;       /* ↓ columna */
  flex-direction: row-reverse;  /* ← reversa */

  /* EJE PRINCIPAL (horizontal en row) */
  justify-content: flex-start;    /* |ABC       */
  justify-content: flex-end;      /*       ABC| */
  justify-content: center;        /*    ABC     */
  justify-content: space-between; /* A   B   C  */
  justify-content: space-around;  /*  A   B   C */
  justify-content: space-evenly;  /* igual a los dos lados */

  /* EJE SECUNDARIO (vertical en row) */
  align-items: stretch;    /* default: se estiran */
  align-items: center;     /* centrado vertical */
  align-items: flex-start; /* arriba */
  align-items: flex-end;   /* abajo */
  align-items: baseline;   /* alineado al texto */

  gap: 16px;          /* espacio entre items */
  flex-wrap: wrap;    /* permite saltos de línea */
}

.item {
  flex-grow: 1;    /* crece para llenar espacio */
  flex-shrink: 0;  /* no se encoge */
  flex-basis: 200px; /* tamaño base */
  flex: 1 0 200px;  /* shorthand: grow shrink basis */
  align-self: center; /* anula align-items solo para este */
  order: 2;           /* reordena visualmente */
}

🔲 CSS Grid

Para layouts en dos dimensiones (filas Y columnas a la vez).

grid demo
Header (span 3)
Sidebar
Main Content
Aside
Footer (span 3)
.contenedor {
  display: grid;

  /* COLUMNAS */
  grid-template-columns: 1fr 2fr 1fr;       /* fracciones */
  grid-template-columns: 200px auto 200px;   /* fijo + auto */
  grid-template-columns: repeat(3, 1fr);    /* 3 iguales */
  grid-template-columns: repeat(auto-fit, minmax(200px, 1fr)); /* responsivo */

  /* FILAS */
  grid-template-rows: 80px auto 60px;

  /* AREAS NOMBRADAS */
  grid-template-areas:
    "header header header"
    "sidebar main aside"
    "footer footer footer";

  gap: 16px;          /* gap entre filas y columnas */
  row-gap: 20px;       /* solo filas */
  column-gap: 10px;    /* solo columnas */
}

.header  { grid-area: header; }
.sidebar { grid-area: sidebar; }

.item {
  grid-column: 1 / 3;      /* de col 1 a col 3 */
  grid-column: span 2;     /* ocupa 2 columnas */
  grid-row: 1 / 3;         /* de fila 1 a fila 3 */
  justify-self: center;    /* alinear en su celda */
}

📍 Posicionamiento

position demo

position: static;    /* default, flujo normal */

position: absolute;  /* relativo al padre con position != static */
position: fixed;     /* fijo al viewport, no scrollea */
position: sticky;    /* normal hasta un punto, luego fijo */

/* Con relative/absolute/fixed/sticky se usan: */

right: 0;
bottom: 0;
left: 50%;

/* Centrar absolutamente */
position: absolute;
top: 50%; left: 50%;
transform: translate(-50%, -50%);

z-index: 10;  /* capa de apilamiento (mayor = encima) */

🌊 Transiciones

demo interactivo

  background: blue;
  transform: scale(1);

  /* TRANSICIÓN */
  transition: all 0.3s ease;
  /* propiedad | duración | timing-function | delay */

  /* Múltiples propiedades */
  transition:
    background 0.3s ease,
    transform 0.2s cubic-bezier(0.34, 1.56, 0.64, 1);
}

.boton:hover {
  background: red;
  transform: scale(1.1) rotate(-2deg);
  box-shadow: 0 8px 20px rgba(255,0,0,0.4);
}

/* TIMING FUNCTIONS */
transition-timing-function: ease;

transition-timing-function: linear;

Animaciones complejas que no dependen de interacción del usuario.

/* 1. Definir la animación */
@keyframes bounce {
  0%, 100% { transform: translateY(0); }
  50%      { transform: translateY(-20px); }
}

@keyframes spin {
  from { transform: rotate(0deg); }
  to   { transform: rotate(360deg); }
}

/* 2. Aplicar al elemento */
.elemento {
  animation-name: bounce;
  animation-duration: 1s;
  animation-timing-function: ease-in-out;
  animation-delay: 0.5s;
  animation-iteration-count: infinite;  /* o un número */
  animation-direction: alternate;        /* normal | reverse | alternate */
  animation-fill-mode: forwards;         /* mantiene estado final */

  /* shorthand */
  animation: bounce 1s ease-in-out infinite;

  /* múltiples animaciones */
  animation: bounce 1s ease infinite, spin 2s linear infinite;
}

🔧 Variables CSS (Custom Properties)

Las variables CSS permiten reutilizar valores y crear temas dinámicos.

mismo botón, distinto --btn-color
/* Definir en :root = disponible globalmente */
:root {
  --color-primario: #ff6b35;
  --color-fondo: #0a0a0f;
  --espacio-md: 16px;
  --radio-btn: 8px;
  --sombra: 0 4px 12px rgba(0,0,0,0.15);
}

.boton {
  background: var(--color-primario);
  padding: var(--espacio-md);
  border-radius: var(--radio-btn);
  box-shadow: var(--sombra);

  /* Valor fallback si la variable no existe */
  color: var(--color-texto, white);
}

/* Sobreescribir solo en un componente */
.tarjeta {
  --color-primario: blue;  /* solo para .tarjeta y sus hijos */
}

/* Cambiar tema con JS */
// document.documentElement.style.setProperty('--color-primario', 'red');

/* Tema oscuro con media query */
@media (prefers-color-scheme: dark) {
  :root { --color-fondo: #111; }
}

📱 Responsive Design

Adaptar el diseño a distintos tamaños de pantalla con media queries.

media queries y unidades responsivas
/* ═══ MEDIA QUERIES ═══ */

/* Mobile first: estilos base para móvil, luego se amplía */
.grid { grid-template-columns: 1fr; }         /* móvil: 1 columna */

@media (min-width: 768px) {                      /* tablet */
  .grid { grid-template-columns: repeat(2, 1fr); }
}

@media (min-width: 1024px) {                     /* desktop */
  .grid { grid-template-columns: repeat(3, 1fr); }
}

/* Por características del dispositivo */
@media (orientation: landscape)   { }  /* modo horizontal */
@media (hover: none)              { }  /* sin mouse (táctil) */
@media print                      { }  /* al imprimir */
@media (prefers-color-scheme: dark){ }  /* modo oscuro del SO */
@media (prefers-reduced-motion: reduce){ } /* sin animaciones */

/* ═══ UNIDADES RESPONSIVAS ═══ */
font-size: 1rem;      /* relativo al root (16px default) */
font-size: 1em;       /* relativo al padre */
width: 50vw;          /* 50% del viewport width */
height: 100vh;        /* 100% del viewport height */
font-size: 5vmin;     /* % del lado más pequeño */
width: min(100%, 800px);     /* el menor valor */
width: max(300px, 50%);      /* el mayor valor */
font-size: clamp(1rem, 3vw, 2rem); /* min, ideal, max */
