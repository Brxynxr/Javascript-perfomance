📄 Estructura base de un documento HTML

Todo archivo HTML comienza con esta estructura. El navegador la necesita para saber cómo interpretar el documento.

código base
<!DOCTYPE html>
<html lang="es">
  <head>
    <meta charset="UTF-8"/>
    <meta name="viewport" content="width=device-width, initial-scale=1.0"/>
    <title>Mi página</title>
  </head>
  <body>
    <!-- Todo el contenido visible va aquí -->
  </body>
</html>

📊 Tablas

Estructura de datos en filas y columnas. Ideal para datos tabulares, no para layouts.

preview
NombreCiudadEdad

JuanMedellín22
Total: 2 registros
 
<table>
  <thead>
    <tr>
      <th>Nombre</th>  <!-- th = encabezado -->
    </tr>
  </thead>
  <tbody>
    <tr>                    <!-- tr = fila -->
      <td>Breyner</td>  <!-- td = celda -->
    </tr>
  </tbody>
  <tfoot>
    <tr><td colspan="3">Pie</td></tr>
  </tfoot>
</table>

<!-- colspan = une columnas, rowspan = une filas -->
<td colspan="2">ocupa 2 columnas</td>

🖼️ Multimedia

  <source media="(min-width:800px)" srcset="grande.jpg"/>

</picture>

<!-- VIDEO -->
<video controls autoplay muted loop poster="thumb.jpg">
  <source src="video.mp4" type="video/mp4"/>
</video>

<!-- AUDIO -->
<audio controls>
  <source src="cancion.mp3" type="audio/mpeg"/>
</audio>

<!-- IFRAME (embebido) -->
<iframe
  src="https://www.youtube.com/embed/VIDEO_ID"
  width="560" height="315"
  frameborder="0"
  allowfullscreen
></iframe>

<!-- FIGURA con pie de foto -->
<figure>
  <img src="foto.jpg" alt="..."/>
  <figcaption>Pie de foto descriptivo</figcaption>
</figure>

🧩 Etiquetas Semánticas

HTML5 introdujo etiquetas con significado. Mejoran el SEO y la accesibilidad.

estructura semántica típica
<header>    <!-- encabezado de página o sección -->
  <nav>     <!-- menú de navegación -->
    <ul><li><a href="#">Inicio</a></li></ul>
  </nav>
</header>

<main>       <!-- contenido principal (único por página) -->
  <article> <!-- contenido independiente (post, noticia) -->
    <section><!-- sección temática dentro del artículo -->
      <h2>Sección</h2>
      <p>Contenido</p>
    </section>
  </article>

  <aside>    <!-- contenido relacionado / sidebar -->
    <p>Contenido secundario</p>
  </aside>
</main>

<footer>     <!-- pie de página -->
  <p>&copy; 2025 Mi sitio</p>
</footer>

<!-- div y span: sin semántica, para estructura/estilos -->
<div class="contenedor">bloque genérico</div>
<span class="highlight">inline genérico</span>

🧠 Meta & Head

Lo que va en el <head> no se ve pero es fundamental para el navegador, SEO y redes sociales.

head completo
<head>
  <!-- Charset y viewport (SIEMPRE primero) -->
  <meta charset="UTF-8"/>
  <meta name="viewport" content="width=device-width, initial-scale=1.0"/>

  <!-- Título y descripción (SEO) -->
  <title>Mi Página | Breyner Dev</title>
  <meta name="description" content="Descripción de 160 chars máximo"/>
  <meta name="keywords" content="html, css, javascript"/>
  <meta name="author" content="Breyner"/>

  <!-- Open Graph (preview en redes sociales) -->
  <meta property="og:title" content="Mi Página"/>
  <meta property="og:description" content="..."/>
  <meta property="og:image" content="thumb.jpg"/>
  <meta property="og:url" content="https://mipagina.com"/>

  <!-- Favicon -->
  <link rel="icon" href="favicon.ico"/>
  <link rel="icon" href="icon.png" type="image/png"/>

  <!-- CSS externo -->
  <link rel="stylesheet" href="styles.css"/>

  <!-- JS externo (defer = carga sin bloquear HTML) -->
  <script src="app.js" defer></script>

  <!-- Fuentes Google -->
  <link rel="preconnect" href="https://fonts.googleapis.com"/>
  <link href="https://fonts.googleapis.com/css2?family=Roboto&display=swap" rel="stylesheet"/>

  <!-- Redireccion automática -->
  <meta http-equiv="refresh" content="5; url=nueva-pagina.html"/>
</head>
